🎸 PrimeSound - Frontend (React + Vite)
📝 Giới thiệu
Giao diện người dùng của PrimeSound được xây dựng nhằm mang lại trải nghiệm nghe nhạc mượt mà như các ứng dụng streaming hiện đại. Tập trung vào tốc độ tải trang và tính thẩm mỹ cao.

👨‍💼 Tư duy thiết kế của Project Manager
Tối ưu hóa hiệu năng: Sử dụng Vite để đảm bảo tốc độ phản hồi (Hot Module Replacement) nhanh nhất trong quá trình phát triển của team.
Quản lý giao diện: Áp dụng Tailwind CSS để xây dựng hệ thống Design System đồng nhất, giúp các thành viên FE phối hợp làm việc mà không bị lệch tông màu hay kích thước.
Cấu trúc dự án: Phân chia thư mục theo tính năng (Feature-based structure) giúp việc quản lý hàng chục nghìn dòng code trở nên dễ dàng, tránh tình trạng "phình" dự án.

✨ Tính năng giao diện
[x] Audio Player: Trình phát nhạc đầy đủ tính năng (Play/Pause, Next/Prev, Volume control).
[x] Discovery UI: Giao diện khám phá âm nhạc trực quan theo danh mục và nghệ sĩ.
[x] Real-time Interaction: Cập nhật trạng thái playlist và yêu thích tức thì.
[x] Responsive Layout: Tối ưu hóa cho mọi thiết bị từ di động đến máy tính.

🛠 Hướng dẫn cài đặt
Đảm bảo đã chạy Backend PrimeSound.
Chạy lệnh:
npm install
npm run dev
